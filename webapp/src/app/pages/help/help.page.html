Tips

All data syncs in real-time — changes show instantly across all logged-in users

Set up document types (Tax Invoice, Proforma, PO, etc.) in Organization > Documents — numbering is automatic

Only the organization owner can edit settings, manage members, or delete the org

GST is auto-calculated on invoices — CGST+SGST for same-state, IGST for inter-state
